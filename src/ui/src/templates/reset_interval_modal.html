Reset the Job Interval?

This job has an interval trigger. Choose one of the buttons below to update when the original job should be run again:
  • Selecting "Yes" updates the next run time of the original job based on the current time.
  • Selecting "No" will keep the original job's existing next run time.
  • Selecting "Cancel" cancels running the job now and no changes will be made to the next run time.
